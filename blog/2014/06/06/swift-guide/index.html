Swift 入门

Jun 6th, 2014

自从苹果公司在 2014 WWDC(Worldwide Developers Conference) 上公布了新的编程语言 Swift 以后，开发者们奔走相告，一时间 Swift 成了大家关注的焦点。本文记录我在了解 Swift 过程中的一些心得。

目前唯一权威的资料是苹果公司在 iBooks 上发布的电子书《The Swift Programming Language》(EPub 格式)，下载地址：微盘、百度网盘。

注：EPub 文件在 Windows 下可以用 Adobe Digital Editions 3.0 查看。

关于 Swift

iOS 或者 Mac OS X 下的应用开发，一直都是使用的 Objective-C（以下简称 OC）。

对于很多有其他语言基础的开发者来说，刚接触到 OC 的语法都相当不适应，无论是函数声明，还是调用，都太颠覆了。

Swift 是一门全新的编程语言。

Swift 采用安全编程模式，并添加了很多现代化的功能特性，使得编程变得更容易、更灵活、更有趣。

《The Swift Programming Language》

Swift 更加类似于传统的 C、C++ 之类的语法，更加简单高效，未来 Swift 必然是 iOS 或 Mac OSX 也能够用开发的首选。

开发环境

Swift 的运行环境，搭建起来很容易，但有一些必备要求：

  • Mac 电脑一台
  • 10.9.3 以上的 Mac OS X 操作系统
  • Xcode 6.0 Beta 版本（目前好像只有付费的开发者能下载成功，下载地址：百度网盘）
