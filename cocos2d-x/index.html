Cocos2D-X 开发

Cocos2D-X引擎介绍

免费开源、国人制作

何为游戏引擎？

引擎，又称为发动机。没有引擎，汽车无法驱动，或者摩托骡拉？坏的引擎，让开发者纠结，玩家虐心。

游戏引擎和交通工具引擎的概念一致，是驱动整体运转的核心部件，是动力的来源。游戏引擎也是产品的核心，引擎好坏决定游戏的品质。

作为游戏的核心，但不是游戏的全部。它是一个内核，提供通用化、标准化的功能，但不提供游戏特定的功能。（Quake游戏和引擎一个名字，这是特例）

引擎跟前端开发中的框架和库的概念也有相似之处。

电子游戏发展初期，没有引擎。1990 DOOM 游戏提出了游戏引擎的概念。早期的游戏，就是多个模块的结合，很难升级和后续维护。引擎开发方式提升了效率，减少了制作成本。

优秀游戏引擎的特征：稳定性、性能（帧率、能够承载的运算量）、功能、简单易用、跨平台。最好还有配套的可视化编辑器和插件。

知名的游戏引擎：虚幻、Unity、Cry Engine、Cocos2D-x、极品飞车、IW

引擎分类：授权引擎、开源引擎、自主引擎

Cocos2D-x是一个支持多平台的手机游戏引擎，使用 C++ 开发，基于 OpenGL ES、DirectX，支持多个平台。

Cocos2D 的名字是因为引擎原创团队所在的阿根廷 Los Cocos，最初是 Python 语言开发的（Python 比较小众），2008 年随着 iPhone 手机的耀世，引擎团队推出了 Cocos2D-iPhone。

2010年11月，Cocos2D-X 第一个版本发布。X，英文 Cross 的缩写，表示交叉，在这里意味着跨平台。Cocos2D-X 的发起者是王哲（Walzer，中国人）

Cocos2D-X 开发的项目是一种特殊的应用程序，但其核心还是应用程序，只不过舍弃了原有平台的 UI 组件，并将原有平台的交互都以消息的形式传递到 Cocos2D-X 的封装架构中，再由开发者通过 Cocos2D-X 的消息响应机制来完成游戏的开发。

Cocos2D-X 特点：免费开源、跨平台（最适合移动平台）
