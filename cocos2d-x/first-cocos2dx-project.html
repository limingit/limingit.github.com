第一个 Cocos2D-X 工程：FirstGame

Apr 29th, 2014

环境：Windows 7 旗舰版(64位) + Cocos2D-X v3.2

使用命令行创建 cocos2d-x 工程

配置 cocos2d-x 的开发环境完成后，就可以创建工程了。

执行命令创建：cocos new -p com.beginios.firstgame -l cpp -d F:\projects\cocos2d-x3 firstgame

Android 工程位于 proj.android 目录下，需要在该目录下执行 python build_native.py 来编译应用。

Eclipse 打开工程

  • 主菜单 File –> New –> Projects… 打开创建工程的对话框
