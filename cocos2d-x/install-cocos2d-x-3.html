安装 Cocos2D-X 3.0

Apr 29th, 2014

2014 年 4月，Cocos2D-X v3.0 正式发布，相对于 v2.2.1，环境的配置又发生了较大的变化，现在记录如下。

第一步是做准备。

如果要开发 Mac 或者 iOS 下的游戏，需要准备一台 Mac 电脑及 iPhone/iPad/iPod Touch之类的设备，真机调试还需要申请苹果的开发者证书。

开发 Windows 游戏或 Android 游戏，有 PC 机和一台 Android 手机/平板就好。

接下来，根据需要下载相关资源：

  • Python Windows 下需要安装，Mac 下已经自带了
  • Cygwin Window 下需要安装虚拟的 Shell 环境，Linux 和 Mac 都不需要
  • Cocos2D-X 3.0
  • [Android SDK] 调试 Android 应用需要
  • [Android NDK] 调试 Android 应用需要
  • [Ant] 调试 Android 应用需要
  • IDE：Mac 下使用 Xcode 就好，Windows 下可以使用 Visual Studio，Android 开发建议使用 Eclipse。

Mac OS X 环境

将 Cocos2D-X 3.0 的压缩包解压以后，在终端上执行命令：python setup.py

source ~/.bash_profile 刷新配置文件

查看命令用法 cocos -h

创建项目

cocos new -h 查看新建命令用法

  • -p 或 --package 指定包名
  • -l 或 --language 指定开发的语言，支持 C++、LUA 和 JS
  • -d 或 --directory 指定项目存放的目录
  • -t 或 --template 指定模板名称
  • --no-native 是 LUA/JS 工程特定参数，指定无需原生支持。

执行命令创建：cocos new -p com.beginios.helloworld -l cpp -d /Users/liming/Documents/projects/cocos2d-x3 helloworld

查看一下刚才创建的目录结构：

  • Classes
  • CMakeLists.txt
  • cocos2d
  • proj.android
  • proj.ios_mac
  • proj.linux
  • proj.win32
  • proj.wp8-xaml
  • Resources

用 Xcode 打开 proj.ios_mac 下的 helloworld.xcodeproj，运行编译即可
